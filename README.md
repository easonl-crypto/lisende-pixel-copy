# AITI Pixel Test

Static copy of the AITI AI model tendency test page.

## Files

- `index.html` redirects to the copied page.
- `pixel.html` contains the page markup.
- `pixel-styles.css` contains the visual styling.
- `pixel-app.js` contains the quiz and poster-generation logic.
- `assets/` contains the local logo, QR code, and persona image assets.
